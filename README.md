**TalentMatch: Resume Scoring & Fit Analyzer**

TalentMatch is a machine learning-based resume scoring and fit analyzer that leverages BERT embedding and cosine similarity to evaluate resumes against job descriptions. 
The system performs **semantic analysis** of keywords, skills, and experience to generate a match score, highlighting areas that are matched or missing. 
It helps job seekers improve their resumes and enables **recruiters to quickly assess candidate fit** using AI-driven insights.

**Features**
- Resume evaluation using BERT embeddings for semantic understanding
- Match scoring based on cosine similarity
- Highlights matched and missing skills or keywords
- Supports data-driven decisions for recruitment

**Technologies**
- **Programming Language**: Python
- **Libraries**: scikit-learn, pandas, numpy
